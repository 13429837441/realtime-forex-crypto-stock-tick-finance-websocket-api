# Error Code Description

| Error Code | Meaning                       |
| ---------- | ----------------------------- |
| 200        | OK                            |
| 400        | Invalid request header param  |
| 400        | Invalid request data param    |
| 401        | Invalid token                 |
| 429        | Rate limit exceeded           |
| 600        | Invalid code                  |
| 601        | Empty body                    |
